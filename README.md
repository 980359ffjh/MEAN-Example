# MEAN App
M - MongoDB

E - Express

A - AngularJS

N - NodeJS

用 MEAN 架構實作簡單的網頁

# 如何使用

先使用 `npm` 下載相關套件

```bash
npm install
```

再使用 `bower` 下載相關套件

```bash
bower install
```

若安裝時出現版本選擇問題
